## Proyectos

### Sistema Bancario
**Archivo**: `sistema-bancario/banco.py`

Sistema de gestión de cuentas bancarias implementando Programación Orientada a Objetos.

**Conceptos aplicados:**
- Clases y objetos (Persona, Cliente)
- Herencia (Cliente hereda de Persona)
- Encapsulamiento (atributos privados)
- Métodos especiales (`__init__`, `__str__`)
- Polimorfismo

**Funcionalidades:**
- Crear cliente con nombre, apellido y número de cuenta
- Depositar y retirar dinero
- Validación de saldo suficiente
- Interfaz por consola con menú

---

### Web Scraping
**Archivo**: `web-scraping/scraping.py`

Extracción automatizada de datos de páginas web.

**Tecnologías:**
- `requests` — Peticiones HTTP
- `BeautifulSoup` — Parseo de HTML
- `lxml` — Parser de XML/HTML

**Conceptos:**
- GET requests a páginas web
- Selección de elementos con selectores CSS
- Extracción de texto e imágenes
- Manejo de estructuras HTML complejas

---

### Facturación Tkinter
**Archivo**: `tkinter-facturacion/mi_restaurante.py`

Aplicación de escritorio para gestión de un restaurante.

**Características:**
- Interfaz gráfica con Tkinter
- Sistema de menús
- Cálculo de totales
- Registro de órdenes

---

### NumPy y Pandas
**Archivo**: `numpy-pandas/introduccion.ipynb`

Notebook de Jupyter con ejercicios de análisis de datos.

**Contenido:**
- Arrays unidimensionales, bidimensionales y tridimensionales
- Atributos de arrays (shape, ndim, dtype, size)
- Operaciones matemáticas con arrays
- Creación de arrays especiales (zeros, ones, random)
- Manejo de errores comunes (broadcasting)
- DataFrames de Pandas
